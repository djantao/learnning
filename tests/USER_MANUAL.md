## 2. 笔记系统

笔记是 MindForge 的知识原子。你在这里记录一切。

### 2.1 写一篇笔记

1. 点击左侧 **📓 笔记本** 进入笔记本列表
2. 选择一个笔记本，点击 **"新建笔记"**（或从侧边栏直接进入）
3. 填写标题，用 Markdown 写内容：

```markdown
# TypeScript 泛型

## 基本概念
泛型让函数和类可以处理多种类型。

## 语法
\`\`\`typescript
function identity<T>(arg: T): T {
    return arg;
}
\`\`\`

## 约束
使用 extends 关键字约束泛型：
\`\`\`typescript
function getLength<T extends { length: number }>(arg: T): number {
    return arg.length;
}
\`\`\`

相关笔记：[[TypeScript基础]] [[设计模式]]
```

编辑器右侧有**实时预览**面板。支持 Ctrl+S 保存，以及 3 秒自动保存。

### 2.2 Wiki 链接 — 建立知识网络

用 `[[笔记标题]]` 语法链接到其他笔记。当你写了 `[[TypeScript基础]]`：

- 如果 "TypeScript基础" 笔记已存在 → 自动创建双向链接
- 如果不存在 → 创建后会链接上
- 打开任意笔记，右侧边栏显示**出链**和**反向链接**

### 2.3 标签

在笔记编辑器的标签输入框中：
- 输入标签名，按**回车**添加
- 点击已有标签上的 **×** 删除
- 标签会出现在搜索结果和标签浏览页中

### 2.4 AI 功能 — 在笔记中使用

编辑笔记时，右侧 AI 面板提供：

**生成闪卡** — AI 自动读取笔记内容，生成 5-10 张用于间隔复习的闪卡。
**生成摘要** — AI 用 2-3 句话概括笔记要点。

建议每写完一篇重要笔记后都用这两个功能。

## 3. 笔记本与章节

### 3.1 创建笔记本

1. 进入**笔记本**页面
2. 点击 **"新建笔记本"**
3. 填写名称（如"前端开发"）、描述、选择图标

  │   ├── 📂 基础语法
  │   │   ├── 📄 变量与类型
  │   │   └── 📄 函数与作用域
  │   └── 📄 异步编程
  ├── 📂 TypeScript
  │   ├── 📄 类型系统
  │   └── 📄 泛型深入
  └── 📄 性能优化

创建章节：在笔记本卡片中，使用**"添加章节"**按钮。

## 4. AI 对话

AI 对话是你 7×24 小时的学习伙伴。它不是普通的 ChatGPT — 它**记住关于你的一切**。

### 4.1 通用对话

1. 点击左侧 **💬 AI 对话** 进入
2. 左侧是对话历史列表，右侧是聊天区
3. 输入问题，按 Enter 发送（Shift+Enter 换行）
4. AI 会流式回复

### 4.2 AI 知道什么

当你和 AI 对话时，系统会自动注入以下上下文：

```
┌─────────────────────────────────┐
│ 🔵 学习档案已加载               │
│   知识水平: 中级                 │
│   学习目标: 成为全栈TS工程师      │
│   偏好风格: 深入原理              │
│                                 │
│ 🟢 指令锚点已注入 (2 条)        │
│   [language] 用简单中文解释      │
│   [depth] 深入讲解底层机制       │
│                                 │
│ 🟡 知识盲区提醒                  │
│   · 泛型理解薄弱 (严重度 75%)    │
│   · 闭包概念模糊 (严重度 60%)    │
└─────────────────────────────────┘
```

所以你不需要每次都说"我是什么水平"、"请用中文"——AI 默认就知道。

### 4.3 课程模式对话

当你在课程学习页面中使用 AI 聊天时，AI 还会额外知道：

- 当前课程的整体进度
- 课程地图（哪些模块已完成/进行中/未开始）
- 当前模块内所有知识点列表
- 同一模块内的薄弱知识点
- 当前知识点的完整内容

这就是**AI 教练模式**——它能精准地在课程上下文中辅导你。

## 5. 课程系统

这是 MindForge 最核心的功能——结构化学习 + AI 教练。

### 5.1 创建课程

1. 点击左侧 **📚 课程** 进入
2. 点击 **"新建课程"**，填写标题（如"Python 入门"）、描述

### 5.2 构建课程结构

课程采用 **课程 → 模块（可3层嵌套）→ 知识点** 的层级结构：

```
🐍 Python入门
  ├── 📘 基础语法
  │   ├── 📘 变量与类型
  │   │   ├── 💡 变量声明
  │   │   ├── 💡 数字类型
  │   │   ├── 💡 字符串操作      ← 当前正在学
  │   │   └── 💡 布尔与None
  │   └── 📘 控制流
  │       ├── 💡 if/elif/else
  │       └── 💡 for/while循环
  ├── 📘 函数与模块
  └── 📘 面向对象
```

在课程详情页：
- 点击 **"添加模块"** → 输入标题
- 点击模块进入 → **"添加子模块"**
- 在子模块中 → **"添加知识点"**，填写标题和内容（Markdown）

### 5.3 开始学习 — 分屏学习视图

点击任意知识点，进入**分屏学习视图**：

```
┌──────────────────────────┬──────────────────┐
│  知识点内容               │  AI 教练         │
│                          │                  │
│  # Python 字符串操作      │  🤖              │
│                          │  我看到你正在学习  │
│  ## 基本操作              │  Python 字符串。   │
│  - 拼接: "Hello "+"World"│  你在"变量与类型"  │
│  - f-string 格式化        │  模块中。          │
│  - 切片: s[0:5]          │                  │
│                          │  推荐你先掌握：     │
│  ## 常用方法              │  "变量声明"和      │
│  .upper() / .lower()     │  "数字类型"还比较   │
│  .split() / .join()      │  薄弱。            │
│  .replace()              │                  │
│                          │  有什么想问的吗？   │
│                          │                  │
│  掌握度: ★★★★☆ (4/5)     │  [输入消息...]    │
│  ← 上一个  下一个 →       │                  │
└──────────────────────────┴──────────────────┘
```

**左侧：** 阅读知识点内容，学完后用星级自评掌握度。
**右侧：** AI 教练聊天——它知道你在学什么、进度如何、哪里薄弱。

### 5.4 AI 复习生成

在知识点学习页，可以请求 AI：

- **生成复习闪卡** — 针对当前知识点生成 5 张闪卡，直接进入复习队列
- **生成费曼问题** — AI 出 3 道问答题，你来回答，AI 评估答案并打分

### 5.5 进度自动传播

当你给知识点打分时：
1. 知识点掌握度更新（1-5 星 → 0-100% 进度）
2. 所在模块进度 = 所有知识点掌握度平均值
3. 父模块进度 = 所有子模块进度平均值
4. 进度跨越 0% → 自动变为"进行中"，达到 100% → 自动变为"已完成"

一切自动级联，你只需学习+评分。

## 6. 间隔复习

基于 SM-2 算法的科学复习系统——在对的时间让你复习。

### 6.1 闪卡来源

闪卡有三个来源：
- **手动创建**：在复习中心手动创建
- **AI 从笔记生成**：在笔记编辑页使用 AI 功能
- **AI 从知识点生成**：在课程学习页使用 AI 复习生成

### 6.2 开始复习

1. 点击左侧 **🔄 复习** → 点击 **"开始复习"**
2. 系统从队列中取出待复习卡片（按难度排序，最难的先来）
3. 每张卡片：
   - 先看问题 → 脑内回忆答案
   - 点击卡片或按**空格键**翻转
   - 对照答案，给自己打分：

```
┌────────────────────────────────────────────┐
│  问题: TypeScript 中 interface 和 type      │
│        的区别是什么？                       │
│                                            │
│  ┌──────────────────────────────────────┐  │
│  │  点击翻转 / 按空格                    │  │
│  └──────────────────────────────────────┘  │
│                                            │
│  [0 完全忘记] [1 几乎忘了] [2 记得一点]     │
│  [3 基本记得] [4 良好 ✨]  [5 完美 🎯]      │
│                                            │
│  快捷键: 1 2 3 4 5 对应评分                 │
└────────────────────────────────────────────┘
```

### 6.3 SM-2 算法做什么

- 评分 ≥ 3（记得）：下次复习间隔**变长**（1天 → 3天 → 7天 → 21天 → ...）
- 评分 < 3（忘了）：间隔**重置为 1 天**
- 每次评分记录详细日志，包括评分前后的 SM-2 状态

算法自动确保你**在快忘记的时候复习**——这是最有效率的时间点。

### 6.4 复习历史

在复习中心 → **复习历史**，可以看到：
- 每次复习会话的日期、时长
- 平均评分
- 评分分布（优秀/良好/一般/忘了）
- 总卡片数 / 待复习数 / 已掌握数

## 7. 学习目标

把零散的学习变成可见的进度。

### 7.1 创建目标树

1. 点击 **🎯 学习目标**
2. 创建顶级目标（如"掌握 TypeScript"）
3. 在目标下创建子目标（如"理解泛型"、"掌握装饰器"）
4. 子目标下还可以继续嵌套

```
🎯 掌握 TypeScript
  ├── 🎯 理解类型系统 (80%)
  │   ├── ✅ 基础类型 (100%)    ← 已完成
  │   ├── 🔄 联合类型 (60%)     ← 进行中
  │   └── ⭕ 类型守卫 (0%)      ← 未开始
  ├── 🎯 理解泛型 (30%)
  └── 🎯 掌握装饰器 (0%)
```

### 7.2 关联学习材料

1. 点击目标 → **"关联材料"**
2. 从已存在的笔记下拉列表中选择
3. 或者粘贴外部资源 URL

### 7.3 自动进度计算

进度是**根据真实学习数据自动计算**的：

| 关联闪卡的复习状态 | 贡献度 |
|-------------------|--------|
| 间隔 ≥ 21 天（已掌握） | 100% |
| 间隔 7-21 天 | 60% |
| 已复习但尚未掌握 | 30% |
| 未复习 | 0% |

你也可以**手动勾选材料完成**作为辅助标记。

目标进度自动级联：子目标进度 → 父目标 = 所有子目标平均值。

## 8. 知识图谱

可视化你的知识网络——看到哪些概念关联在一起。

1. 点击 **🕸️ 知识图谱** 进入
2. 每个**节点**代表一篇笔记，大小反映链接数量
3. 节点间的**连线**代表 Wiki 链接关系
4. 操作：
   - **滚轮缩放** | **拖拽平移** | **悬停看标题**
   - **点击节点** → 直接跳转到那篇笔记

```
     [TypeScript基础] ──── [TypeScript泛型]
          │                    │
          ├── [设计模式] ──────┘
          │
          └── [React开发]
               │
               └── [状态管理]
```

图谱让零散的笔记变成有结构的**知识网络**。随着你写更多笔记和 Wiki 链接，图谱会越来越有价值。

## 9. 搜索与标签

### 9.1 全文搜索

1. 点击 **🔍 搜索** 进入
2. 输入关键词 → 300ms 防抖后自动搜索
3. 搜索范围：笔记的**纯文本内容**
4. 结果按更新时间排序，显示标签和摘录

### 9.2 标签浏览

1. 点击 **🏷️ 标签** 进入
2. 左侧标签列表，右侧该标签下的笔记
3. 点击标签过滤笔记
4. 搜索结果也支持按标签筛选

## 10. 学习档案

让 AI 更了解你。

### 10.1 配置学习档案

点击 **⚙️ 设置**，进入**学习档案**卡片：

| 配置项 | 说明 | 示例 |
|-------|------|------|
| 学习目标 | 你的整体学习方向 | "成为全栈 TypeScript 工程师" |
| 知识水平 | 当前水平 | 初级 / 中级 / 高级 |
| 偏好风格 | AI 讲解风格 | "深入原理、注重实践" |
| 语言偏好 | AI 回复语言 | 中文 / 英语 / 中英混合 |
| 深度偏好 | 解释详细程度 | 深入 / 适中 / 简洁 |

### 10.2 指令锚点

指令锚点是给 AI 的**硬性规则**——AI 每次对话都会遵守：

```
┌──────────────────────────────────────┐
│ 锚点 1: 简单中文                     │
│ 分类: language                       │
│ 内容: "请用简单易懂的中文解释，        │
│        避免术语堆砌"                  │
│                                      │
│ 锚点 2: 深度讲解                     │
│ 分类: depth                          │
│ 内容: "如果涉及原理，请深入讲解        │
│        底层机制"                      │
└──────────────────────────────────────┘
```

添加/删除锚点后点击保存，下次 AI 对话立即生效。

### 10.3 知识盲区

系统会定期分析你的复习记录，自动检测薄弱领域。
在 AI 对话中，AI 会特别关注这些盲区，帮你针对性强化。

## 11. 典型使用场景

### 场景 A：读完一本书，内化知识

```
1. 创建笔记本 "《代码整洁之道》"
2. 每读完一章，写一篇笔记（Markdown 摘录+自己的理解）
3. 用 [[Wiki链接]] 连接相关概念
4. 对每篇笔记 → AI 生成闪卡 → AI 生成摘要
5. 每天打开复习中心 → 完成当日复习
6. 在知识图谱中看到知识结构可视化
```

### 场景 B：系统学习一门新技术

```
1. 创建课程 "Rust 入门"
2. 按章节建立模块树 → 填入知识点
3. 每天学习 2-3 个知识点：
   a. 左侧阅读内容
   b. 右侧与 AI 教练讨论
   c. 自评掌握度
   d. AI 生成复习闪卡
4. 创建学习目标 → 关联笔记 → 跟踪进度
5. 定期回顾仪表盘 → 看进度和盲区提醒
```

### 场景 C：备考复习

```
1. 将备考资料整理为笔记（用 Wiki 链接串联概念）
2. 为每篇笔记生成闪卡
3. 每天用间隔复习刷卡片
4. 对薄弱知识点 → 进入课程学习页 → AI 教练深入讲解
5. 设定考试日期为目标截止日期 → 跟踪整体进度
6. 搜索功能快速定位遗忘的知识点
```

### 场景 D：日常知识管理

```
1. 读到好文章 → 写笔记摘录 + 自己的思考
2. 开会学到的 → 写笔记记录
3. 碎片想法 → 快速写笔记
4. 用标签分类：前端 / 后端 / 设计 / 管理
5. 周末浏览知识图谱 → 发现笔记间的新关联
6. 和 AI 对话中引用笔记 → AI 帮你关联和深化
```
